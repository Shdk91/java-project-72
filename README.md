<p>If you want to start app local you need start with environment variable APP_ENV = development </p>

# Start

```sh
make
```

## Setup
```sh
make build
```

## Run
```sh
make run
```

## Run tests
```sh
make test
```

## Run checkstyle
```sh
make lint
```
